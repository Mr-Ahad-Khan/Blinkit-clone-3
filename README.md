# 🛒 Blinkit Clone

A modern and responsive **Blinkit-inspired grocery shopping application** built with **React**, **Tailwind CSS**, and **Vite**. The project recreates the core Blinkit shopping experience with product categories, flash deals, search functionality, shopping cart, and responsive layouts using reusable React components.

---

# ✨ Features

- 🛒 Responsive Grocery Shopping Interface
- 🔍 Product Search
- ⚡ Flash Deals Section
- 🏷️ Product Categories
- 🛍️ Product Listings
- ➕ Add to Cart Functionality
- 💳 Shopping Cart
- 🎯 Discount & Offer Badges
- 📱 Mobile-First Responsive Design
- 🎨 Modern UI with Tailwind CSS
- ♻️ Reusable React Components
- ⚡ Fast Performance with Vite

# 🚀 Getting Started

## Clone the Repository

```bash
git clone https://github.com/Mr-Ahad-Khan/Blinkit-Clone.git
```

## Navigate to the Project

```bash
cd Blinkit-Clone
```

## Install Dependencies

```bash
npm install
```

## Run the Development Server

```bash
npm run dev
```

Open your browser and visit:

```text
http://localhost:5173
```
